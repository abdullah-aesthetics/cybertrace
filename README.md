# CyberTrace v2.0
## Hybrid OSINT & Network Forensic Intelligence Platform

A web-based defensive cybersecurity platform for investigating suspicious IPs, domains, file hashes, URLs, and emails.

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```

Open http://localhost:5000

## Deploy on Railway

1. Push this repo to GitHub
2. Go to railway.app
3. New Project → Deploy from GitHub
4. Select this repo
5. Generate Domain → Done!
